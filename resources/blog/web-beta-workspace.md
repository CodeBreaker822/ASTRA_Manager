## Online first

JERVA Web is designed for the server-hosted workflow. Audio is uploaded or recorded in the browser, then processed by the configured server providers.

The browser does not ship offline Whisper models, VAD, diarization, or device resource controls. Those features remain part of the desktop edition.

## What beta includes

- Account registration and secure authentication
- A JERVA-themed web interface
- Pricing and entitlement configuration
- Marketing pages for product, pricing, blog, and desktop downloads

## What comes next

The workspace phase adds persistent transcript projects, upload transcription, live chunk ingestion, polish, summarize, export, processing logs, and quota enforcement.
